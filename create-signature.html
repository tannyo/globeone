Notes

Highlight the interior of the Signature Card, right click and select Copy. Go into your Outlook client and Paste the signature into a new signature. Set the signature as your default signature for GlobeOne.

If you are using the Mac Mail application, please contact desktop support as the signature has to be added manually.

Detailed Instructions

If you are using Mac Mail

  1. Click the “Copy Signature for Mac Mail” button above.

  2. Click in the signature source code field above.
  3. Press Command-A to highlight the signature source code.
  4. Press Command-C to copy the signature source code.
  5. Open Mac Mail.
  6. Open Preferences and select Signatures. Command-,
  7. Create a new signature. This will create a placeholder signature file that will be used later.
  8. Name it GlobeOne.
  9. Drag the signature to Exchange in the first column.

  10. Click Exchange.
  11. Under the Choose Signature dropdown, select GlobeOne.
  12. Close the Preferences dialog. Command-W
  13. Quit Mac Mail. Command-Q
  14. Open Finder.

  15. Press Command-Shift-G.
  16. Enter ~/Library/Mail/V2/MailData/Signatures

  17. Click the Go button.
  18. Press the List button.
  19. Click the Date Modified column.
  20. Click on the .mailsignature file with the latest date.

  21. Right click the file at the file icon.
  22. Select Open With -> Other.

  23. Type textedit until the TextEdit app appears.
  24. Click the Open button or press the return key.

  25. Select and delete everything between the <body></body> elements.

  26. Go back to the signature creator web page, make sure you have filled out all fields and the Signature Card is correct.
  27. Click the "Copy Signature to Clipboard for Mac Mail” button.
  28. Highlight the html under the Signature Card. The easiest way to do this is to click on the html, press Command-A, then Command-C.
  29. Go back to TextEdit.
  30. Paste the signature between the <body></body> elements. Command-V

  31. Save the file with Command-S.
  32. Close the file and quit TextEdit. Command-W then Command-Q.
  33. Go back to Finder.
  34. Right click the file and click on Get Info. Command-I.

  35. The file info dialog will appear.
  36. Click the Locked checkbox.

  37. Close the dialog.
  38. Open Mac Mail.

Create a new email message. Make sure you have selected your GlobeOne email address. Your new signature should appear at the bottom of the email message.

If you are using Mac Outlook Mail

  1. With the mouse, highlight the created signature.

  2. Press Command-C to copy the signature
  3. Open Outlook
  4. Click the Outlook menu, then Preferences. Command-,

  5. Click Signatures

  6. Click the plus (+) button to add a new signature.

  7. Delete the contents of the Signature field and paste (Command-V) in your new signature.

  8. Rename the signature to GlobeOne by clicking the signature name, changing the name, then pressing the Return key.

  9. Click the “Default Signatures...” button.

  10. Click the Globeone account and select the GlobeOne signature, then click the “OK” button.

  11. Click the close dialog button (red dot) to close preferences.

Create a new email message. Your new signature should appear at the bottom of the email message.

If you are using Windows Outlook Mail

  1. With the mouse, highlight the created signature.

  2. Press <Ctrl-C> to copy the signature
  3. Open Outlook
  4. Click the File menu, then Options.

  5. Click Mail, then the “Signatures…” button

  6. Click the “New” button to add a new signature.

  7. Name the signature “GlobeOne” and click the “OK” button.

  8. Paste the signature <Ctrl-V> into the signature editor.

  9. Click the New messages drop down and select GlobeOne.

  10. Click the Replies/forwards drop down and select GlobeOne, then click the “OK” button.

  11. Click the “OK” button to close Outlook Options.

Create a new email message. Your new signature should appear at the bottom of the email message.

If you are using Web Outlook Mail

  1. With the mouse, highlight the created signature.

  2. Press <Ctrl-C> or Command-C to copy the signature
  3. Open the Outlook Web App
  4. Click the gear icon, then Options.

  5. The options page will display.
    1. Click “Email signature”
    2. Paste the signature <Ctrl-V> or Command-V into the signature editor.
    3. Click “Original”
    4. Check “Automatically include my signature on messages I send”
    5. Click “Save”
    6. Click “Options” () to close Outlook Options.

Create a new email message. Your new signature should appear at the bottom of the email message.
